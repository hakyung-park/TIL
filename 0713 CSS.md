### 문법

- Class selector
  - 여러군데에서 사용이 가능함

```css
.클래스명 {
    속성1: 속성값1;
    속성2: 속성값2;
}
```

- Tag selector

```css
태그 {
    속성1: 속성값1;
    속성2: 속성값2;    
}
```

- ID selector
  - 딱 한 번만 쓸 수 있음
  - 하나의 element를 unique하게 찝을 때 사용

```css
#아이디명 {
    속성1: 속성값1;
    속성2: 속성값2;   
}
```





### 예시

```css
.myStyle {
    color: red;
    background-color: yellow;
    font-size: 15px;                
}
```

```css
h1{
    color: blue;
}
```

```css
#myId {
    color: violet;
}
```
